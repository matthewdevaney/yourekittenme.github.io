What Is A Static Website and Why Build One With Pelican?

Pelican is a static website generator powered by Python. Why pelican

The benefits of creating a static website are many:

Fast: static websites only serve content that was already rendered at the time of upload.

Cheap: static websites are less expensive to host because all they require to run is a basic web server, file storage space, and they do not require a database.

Security: static websites are less vulnerable to attack because there is no database.

Easy: No advanced programming knowledge is required to build a static website. In most cases a basic understanding of HTML, CSS and the native language of the website generator will do.
